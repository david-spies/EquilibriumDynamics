The core paradox

Integrating a Neural ODE to a large fixed time T is expensive — the solver makes O(T) function evaluations. But finding the steady state (t → ∞) costs only O(1) in the time dimension. Solving for less gets you there faster.

The three acceleration techniques

1 Steady-state formulation

Instead of integrating dh/dt = f(h,t,θ), we define a loss on the fixed point h* where the dynamics vanish: f(h*, θ) = 0. The equilibrium IS the answer.

Find h* such that f(h*, θ) = 0
2 Root-finding forward pass (Broyden's method)

Quasi-Newton solvers like Broyden's method find h* directly in log-linear iterations — no step-by-step ODE integration required. This replaces the ODE solver's forward pass entirely.

h*n+1 = h*n − J⁻¹ · f(h*n, θ) [quasi-Newton update]
3 Implicit differentiation (no adjoint integration)

Using the Implicit Function Theorem, gradients flow through the fixed-point equation algebraically. No backprop through solver steps, no vanishing gradients across long trajectories.

∂h*/∂θ = −(∂f/∂h*)⁻¹ · ∂f/∂θ

Gauß–Legendre quadrature speed-up (non-steady systems)

When a steady state doesn't exist, the reverse-time adjoint integral can be approximated in parallel using Gaussian quadrature nodes, outperforming sequential RK4/Dopri5 integrators by evaluating multiple time points simultaneously.

Problem: solve dy/dx + 2xy = 0, y(0) = 1

The exact solution is y(x) = e−x². A PINN approximates this without seeing any labelled (x, y) pairs — only the ODE residual and the initial condition act as supervision.

Loss function anatomy

Physics loss (PDE residual) collocation pts
L_pde = 𝔼x[ (dy/dx + 2xy)² ]

Computed via automatic differentiation. Penalises any predicted ŷ(x) that doesn't satisfy the ODE. No labelled data needed — only x-coordinates (collocation points).

Initial condition loss boundary pt
L_ic = (ŷ(0) − 1)²

One scalar constraint anchoring the solution to the known initial condition y(0) = 1.

Total loss
L = L_pde + L_ic

Why no labelled data?

The key insight: the ODE itself provides supervision everywhere in the domain. Auto-differentiation computes ∂ŷ/∂x exactly from the network's computation graph, so the residual dy/dx + 2xy can be evaluated at any x without knowing the true y(x). The network learns to satisfy the equation, not to fit labelled points.

Three key advantages

Mesh-free
Operates directly on scattered sensor readings — no 3D grid or mesh generation needed. Works with irregular, sparse, or moving observation points.
Operator learning
Learns the solution operator (initial condition → trajectory), not just one solution. After offline training, predicts new ICs in a single forward pass — no re-solving.
Lyapunov stability
Finite-time Lyapunov Exponent regularisation penalises chaotic divergence in the latent ODE. Prevents exponential error growth in long-horizon predictions.

Training vs inference workflow

Offline training
1. Encode sensor data → z₀
2. Evolve z via Neural ODE
3. Physics head computes residuals
4. Decode z(t) → solution field
5. Backprop all four components jointly
Online inference
1. Encode new IC → z₀
2. Single forward pass through ODE layer
3. Decode → full solution field

No solver calls at inference time
